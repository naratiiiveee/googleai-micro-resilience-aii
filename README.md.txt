# MicroResilience AI 🌱

## Proje Hakkında
İklim krizinin tarım üzerindeki etkilerini hafifletmek amacıyla geliştirilmiş, toprak ve bitki mikrobiyomunu analiz eden yapay zeka destekli bir web uygulamasıdır. 

## Problem
İklim değişikliği (kuraklık, artan tuzluluk vb.), tarım alanlarındaki toprak mikrobiyom dengesini bozmakta ve bitki verimini düşürmektedir. Çiftçiler ve araştırmacılar, toprağın mikrobiyal sağlığını ve strese karşı direncini hızlı ve etkili bir şekilde analiz edecek uygun maliyetli araçlara ihtiyaç duymaktadır.

## Nasıl Çalıştırılır?
Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1. Depoyu klonlayın: `git clone https://github.com/kullaniciadi/microresilience-ai.git`
2. Gerekli kütüphaneleri yükleyin: `pip install -r requirements.txt`
3. Uygulamayı başlatın (Streamlit kullanıyorsanız): `streamlit run features/app.py`
